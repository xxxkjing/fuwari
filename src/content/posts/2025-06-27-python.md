# 需求

想要快速将一个大文件从一个电脑传输到另一个电脑

如果使用U盘，需要传两次；使用奶牛快传，有大小限制；FTP和SMB服务都需要登录

# 解决

在Python中自带一个模块 \`http.server\` ，很简陋的一个http服务模块，但是传输文件足够了。

在终端中执行：

```bash
python-3.12.4-embed-amd64\python.exe -m http.server 1145
```
